# Sesy Queen 🛒
A fully functional e-commerce kitchenware store built with PHP, MySQL and JavaScript.

## Features
- User authentication (register, login, forgot/reset password)
- Product catalogue with multi-image support
- Cart, wishlist & checkout system
- Order tracking with live status updates
- Email notifications to customer & admin
- Full admin dashboard
- Dark mode & fully responsive

## Tech Stack
PHP · MySQL · JavaScript · Bootstrap Icons · Brevo API

## Note
`config.php` is excluded from this repo as it contains API keys and database credentials.
To run locally, create your own `config.php` with your database and Brevo settings.
